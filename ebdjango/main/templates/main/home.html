Welcome

This is a web app designed to create a shopping list from a set of recipes selected.


The button allows you to create a new recipe by typing the name of the new recipe.


The button displays all the saved recipes and the number of people eating that recipe. By clicking on the name of each recipe, the ingredients and preparation are displayed.


The button displays all the ingredients required for the people you selected in each recipe.
